# TypeScript Project

This repository contains TypeScript exercises and examples for learning and practicing TypeScript concepts, including interfaces, classes, and advanced typing.

## Structure

- `task_1`: Basic TypeScript interfaces and classes.
- `task_4`: Advanced subjects and teacher management using TypeScript.

## Usage

Clone the repository and run the TypeScript files using `ts-node` or compile them with `tsc`.
